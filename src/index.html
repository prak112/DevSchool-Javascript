Breakpoints & Containers

Breakpoints are used to create responsive content, i.e, content layout changes based on the device-width (viewport size).

Content in the container responds as follows :

  • IF device-width or viewport size > min-width, THEN content EXPANDS
  • IF device-width or viewport size < min-width, THEN content SHRINKS

The above content is in a default container, which adapts similar to container-sm
